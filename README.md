# My electron starter kit

- React
- TypeScript
- webpack
- Font Awesome
- mocha
- enzyme
- electron-builder
- sqlite3
- CI
  - AppVeyor (for windows)
  - Travis CI (for macOS)
  - CircleCI (for Linux)

## Development

```
# start electron application
$ yarn run start

# run webpack process
$ yarn run watch

# run lint
$ yarn run lint

# run test
$ yarn run test
```
